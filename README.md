# 🛡️ Reliability AI — Automated Model Evaluation & Drift Shield

An end-to-end MLOps project that monitors production ML models, detects data drift, and uses GenAI to generate incident reports.

## 📋 Features
- ✅ **Baseline Model Training (Customer Churn Classification)
- ✅ **Data Drift Detection (KS Test, PSI)
- ✅ **Performance Monitoring (ROC-AUC, Precision, Recall)
- ✅ **GenAI Incident Responder
- ✅ **Web Dashboard
- ✅ **Production Ready Deployment Files

## 🚀 Quick Start
```bash
# Install dependencies
pip install -r requirements.txt

# Run the app locally
python app.py
```

## 📊 How It Works
1. Train a baseline churn model on historical customer churn model
2. Simulate production data with optional drift
3. Monitor drift and performance
4. Auto-generate incident reports

## 🛠️ Tech Stack
- Python, Flask
- Scikit-learn, XGBoost
- OpenAI
- Tailwind CSS
- Render (for deployment)
